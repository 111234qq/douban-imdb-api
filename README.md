douban-imdb-api
---------------
一个基于豆瓣、`IMDB`、烂番茄评分的电影/电视剧双语(中英)数据`api`接口，根据豆瓣电影/电视剧`id`，可直接获取其主演、导演、类型、编剧、评分、海报等全部中英文数据。

使用示例
----
### 接口地址

    #后面的23333为豆瓣电影/电视剧id
    https://www.querydata.org/api/movie?id=23333

### 接口演示
